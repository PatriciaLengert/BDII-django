# Para rodar o projeto:

## Crianção de uma venv
  - python -m venv venvbd
## Ativar venv
  - .\venvBD\Scripts\activate.ps1
  - LINUX - source venvBD/bin/activate

## Instalar as LIBS
### Django
  - pip install django

### pymysql
  - pip install mysqlclient

## Rodar o projeto
  - python manage.py runserver
